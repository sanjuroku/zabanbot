Features
你能用它做什么？
💬

AI 对话

基于 GPT-4.1，支持多轮对话、图片分析、翻译。说「咋办」自动触发，个性角色可自定义。

🔮

塔罗 · 运势

每日占卜神社签 + 塔罗牌，精美图卡渲染。连签有惊喜！

🎰

扭蛋机

用咋办币抽奖，SSR / SR / R 三稀有度，十连保底 SR。收集完整图鉴，碎片可兑换。

🎬

书影音游

整合 NeoDB 与 Bangumi 搜索书籍、动画、游戏与音乐，带评分和封面。Steam 价格查询，萌娘百科 infobox 截图，一站搞定。

🐾

可爱 & 艺术

从 Reddit 随机抽取可爱动物图片 / 视频（NSFW 自动过滤），或欣赏随机艺术品。

🛠️

实用工具

图片格式转换、读音查询（拼音 / 假名 / IPA）、世界时钟、投票、冷知识、知识问答……
